# Fraud Bill Detection

## Project Description
The Fraud Bill Detection project aims to develop a comprehensive solution for identifying fraudulent bills through image classification techniques. The system utilizes advanced Optical Character Recognition (OCR) combined with machine learning algorithms to analyze and classify images of bills in two stages:

1. **First Stage**: Utilize the CG1050 dataset for initial image classification.
2. **Second Stage**: Implement the Kaggle OCR dataset for more refined image classification.

## Installation Instructions
To set up the project locally, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Adityarajj23/Fraud_bill_detection.git
   cd Fraud_bill_detection
   ```
2. **Install the required libraries**:
   Use the following command to install the necessary Python packages:
   ```bash
   pip install -r requirements.txt
   ```
3. **Prepare the dataset**:
   Follow the instructions below to download the datasets.

## Dataset Download Instructions
### Kaggle OCR Dataset (2nd Stage)
1. Visit the [Kaggle OCR dataset link](https://www.kaggle.com/datasets/) to access the dataset.
2. You will need a Kaggle account to download the dataset. If you don't have one, please sign up.
3. Once signed in, navigate to the specific dataset page and click on the "Download" button.
4. After downloading, extract the files into the `data/ocr` directory in your project folder.

### CG1050 Dataset (1st Stage)
1. Access the CG1050 dataset by following this [link](https://example.com/cg1050-dataset) (replace with the actual link).
2. Download the dataset files and unzip them.
3. Place the extracted files into the `data/cg1050` directory in your project.
